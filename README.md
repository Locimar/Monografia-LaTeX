Monografia-LaTeX (Unochapecó)
=============================

Template LaTeX do curso de ciência da Computação (Unochapecó)

## Como usar?
```
git clone git@github.com:danielgirotto/Monografia-LaTeX.git 
cd Monografia-LaTeX
pdflatex monografia.tex ; okular monografia.pdf
```
